Git init : initialises the folder and create a new .git folder
to see the git changes : git status
if you have changed something... save them using : git add . 
and then commit to save them : git commit -m "version_name"
add a username : git config --global user.name "gyana"
add a email : git config --global user.email "gyanendramahanta8@gmail.com"
to see the email of git account : git config user.email
To see all the logs : git log
How to create a new branch or change branch : git checkout -b "branch name"
To push changes : git push -u origin <branch name>
If you are changed something and not add : git diff
To clone git : git clone <ssh key here>
Add git branch : git branch <branch name>
Show branches : git branch
